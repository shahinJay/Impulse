# Impulse

**Orbital Dynamics And Mission Control Simulator**

### ----🚧🚧🚧🚧 UNDER CONTRUCTION 🚜👷🏗️🚧🚧🚧🚧----

A from-scratch, scriptable orbital mechanics simulator built in C++ with Raylib. Designed for mission planning, trajectory visualization, and delta-v budgeting.
